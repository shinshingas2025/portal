UpdateUser 預存程序

說明：

此預存程序會更新資料庫中的使用者設定。輸入參數包括「使用者識別碼」、「電子郵件地址」及「密碼」。

定義： 
    
   CREATE PROCEDURE Portal_UpdateUser
   (
       @UserID        int,
       @Email           nvarchar(100),
       @Password    nvarchar(50)
   )
   AS
   
   UPDATE Portal_Users
   
   SET
       Email    = @Email,
       Password = @Password
   
   WHERE
       UserID    = @UserID
使用的資料庫資料表：

Users：Users 資料表中的每項記錄都是唯一使用者識別項。此資料表的主索引鍵為 UserID 識別欄位。
