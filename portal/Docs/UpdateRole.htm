UpdateRole 預存程序

說明：

此預存程序由 Roles 系統管理頁使用，以更新指定角色的顯示名稱。輸入參數為 RoleID。

定義： 

   CREATE PROCEDURE Portal_UpdateRole
   (
       @RoleID      int,
       @RoleName    nvarchar(50)
   )
   AS
   
   UPDATE Portal_Roles
   
   SET
       RoleName = @RoleName
   
   WHERE
       RoleID = @RoleID
使用的資料庫資料表：

Roles：Roles 資料表中的每項記錄會在選取的入口網站中定義一項唯一角色，並透過 UserRoles 查詢值資料表，將使用者加入角色。

此資料表的主索引鍵為 RoleID 識別欄位。
