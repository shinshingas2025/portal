GetLinks 預存程序

說明：

此預存程序傳回入口網站內特定 Links 模組的所有連結。輸入參數為 ModuleID。只有到期日晚於今天的宣告才會顯示。

定義： 
    
   CREATE PROCEDURE Portal_GetLinks
   (
       @ModuleID int
   )
   AS
   
   SELECT
       ItemID,
       CreatedByUser,
       CreatedDate,
       Title,
       Url,
       ViewOrder,
       Description
   
   FROM Portal_Links
   
   WHERE
       ModuleID = @ModuleID
   
   ORDER BY
       ViewOrder
使用的資料庫資料表：

Links：如 Links Portal 模組所示，Links 資料表中的每筆記錄都是超連結。由於所有的 Links 模組會將其記錄存放在此資料表中，因此每個項目都包含 ModuleID 以容許在單一查詢中擷取相關項目。

此資料表的主索引鍵為 ItemID 識別欄位。請注意連結說明不得超過 2000 個字元。
