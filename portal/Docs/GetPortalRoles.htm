GetPortalRoles 預存程序

說明：

此預存程序傳回指定入口網站的所有安全性角色。輸入參數為 PortalID。

定義： 
        
   CREATE PROCEDURE Portal_GetPortalRoles
   (
       @PortalID  int
   )
   AS
   
   SELECT  
       RoleName,
       RoleID
   
   FROM Portal_Roles
   
   WHERE   
       PortalID = @PortalID
使用的資料庫資料表：

Roles： Roles 資料表中的每項記錄會在選取的入口網站中定義一項唯一角色，並透過 UserRoles 查詢值資料表，將使用者加入角色。

此資料表的主索引鍵為 RoleID 識別欄位。
