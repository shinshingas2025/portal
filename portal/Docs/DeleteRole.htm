DeleteRole 預存程序

說明：

此預存程序從資料庫刪除選擇的角色。輸入參數為 RoleID，也就是記錄的主索引鍵。

定義： 
    
   CREATE PROCEDURE Portal_DeleteRole
   (
       @RoleID int
   )
   AS
   
   DELETE FROM Portal_Roles
   
   WHERE
       RoleID = @RoleID
使用的資料庫資料表：

Roles： Roles 資料表中的每筆記錄在選擇的入口網站中會定義一個唯一的角色，UserRoles 查詢值資料表會將使用者加入角色。

此資料表的主索引鍵為 RoleID 識別欄位。
