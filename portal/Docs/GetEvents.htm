GetEvents 預存程序

說明：

此預存程序傳回入口網站內特定 Events 模組的所有事件。輸入參數為 ModuleID。只有到期日晚於今天的事件才會顯示。

定義： 
		
   CREATE PROCEDURE Portal_GetEvents
   (
       @ModuleID int
   )
   AS
   
   SELECT
       ItemID,
       Title,
       CreatedByUser,
       WhereWhen,
       CreatedDate,
       Title,
       ExpireDate,
       Description
   
   FROM Portal_Events
   
   WHERE
       ModuleID = @ModuleID
     AND
       ExpireDate > GetDate()
使用的資料庫資料表：

Events：如 Events Portal 模組所示，Events 資料表中的每筆記錄都是單一項目。由於所有的 Events 模組都將其記錄存放在此資料表中，因此每個項目都包含 ModuleID 以容許在單一查詢中擷取相關項目。

此資料表的主索引鍵為 ItemID 識別欄位。請注意說明事件不得超過 2000 個字元。
