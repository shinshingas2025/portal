UserLogin 預存程序

說明：

此預存程序會比對 Users 資料表的認證儲存，對使用者進行驗證。如果找到電子郵件地址/密碼的組合，便會傳回使用者的 UserName。

定義： 
        
   CREATE PROCEDURE Portal_UserLogin
   (
       @Email    nvarchar(100),
       @Password nvarchar(50),
       @UserName nvarchar(100) OUTPUT
   )
   AS
   
   SELECT
       @UserName = Name
   
   FROM Portal_Users
   
   WHERE
       Email = @Email
     AND
       Password = @Password
使用的資料庫資料表：

Users： Users 資料表包含可用於表單架構驗證的使用者認證資訊。
