GetRoleMembership 預存程序

說明：

此預存程序會傳回屬於指定入口網站安全性角色成員的所有使用者。輸入參數為 RoleID。

定義： 
    
   CREATE PROCEDURE Portal_GetRoleMembership
   (
       @RoleID  int
   )
   AS
   
   SELECT  
       Portal_UserRoles.UserID,
       Name,
       Email
   
   FROM Portal_UserRoles
       
   INNER JOIN 
       Portal_Users On Portal_Users.UserID = Portal_UserRoles.UserID
   
   WHERE   
       Portal_UserRoles.RoleID = @RoleID
使用的資料庫資料表：

UserRoles：UserRoles 資料表提供入口網站安全性角色 (定義於 Roles 資料表) 與使用者 (定義於 Users 資料表) 之間的多對多連接。在使用 UserRoles 資料表時，每名使用者都能屬於多種角色，而且每項角色的成員也可以為多名使用者。

UserRoles 資料表沒有主索引鍵。

Users：Users 資料表中的每項記錄都是唯一使用者識別項。此資料表的主索引鍵為 UserID 識別欄位。
