UpdateLink 預存程序

說明：

此預存程序可由 Links 編輯頁使用，以套用變更至現有的連結項目。輸入參數包括項目的主索引鍵 (ItemID)，以及標題、URL 與說明。編輯者的 UserName 也會傳入，以便更新 CreatedByUser 欄位。

定義： 

   CREATE PROCEDURE Portal_UpdateLink
   (
       @ItemID      int,
       @UserName    nvarchar(100),
       @Title       nvarchar(100),
       @Url         nvarchar(250),
       @MobileUrl   nvarchar(250),
       @ViewOrder   int,
       @Description nvarchar(2000)
   )
   AS
   
   UPDATE Portal_Links
   
   SET
       CreatedByUser = @UserName,
       CreatedDate   = GetDate(),
       Title         = @Title,
       Url           = @Url,
       MobileUrl     = @MobileUrl,
       ViewOrder     = @ViewOrder,
       Description   = @Description
   
   WHERE
       ItemID = @ItemID
使用的資料庫資料表：

Links：如 Links Portal 模組所示，Links 資料表中的每筆記錄都是超連結。由於所有的 Links 模組會將其記錄存放在此資料表中，因此每個項目都包含 ModuleID 以容許在單一查詢中擷取相關項目。

此資料表的主索引鍵為 ItemID 識別欄位。請注意連結說明不得超過 4000 個字元。
