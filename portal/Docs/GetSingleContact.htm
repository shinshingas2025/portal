GetSingleContact 預存程序

說明：

此預存程序從資料庫傳回單一連絡人項目。輸入參數為 ItemID，也就是記錄的主索引鍵。傳回的值包括連絡人詳細資料，以及建立與最後編輯此項目的使用者相關資訊。

此預存程序可由 Contacts 的編輯頁使用。

定義： 
    
   CREATE PROCEDURE Portal_GetSingleContact
   (
       @ItemID int
   )
   AS
   
   SELECT
       CreatedByUser,
       CreatedDate,
       Name,
       Role,
       Email,
       Contact1,
       Contact2
   
   FROM Portal_Contacts
   
   WHERE
       ItemID = @ItemID
使用的資料庫資料表：

Contacts： 如 Contacts Portal 模組所示，Contacts 資料表中的每筆記錄都是單一項目。由於所有的 Contacts 模組都將其記錄存放在此資料表中，因此每個項目都包含 ModuleID 以容許在單一查詢中擷取相關的項目。

此資料表的主索引鍵為 ItemID 識別欄位。
