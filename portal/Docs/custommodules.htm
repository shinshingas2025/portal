ASP.NET 入口網站入門套件文件
 

建立自訂入口網站模組
入口網站模組會組合一些程式碼及使用者介面，讓使用者看到特定的功能 (例如：討論回應)，或是轉換資料、圖形與文字 (例如：「區域銷售」報告)。此外，入口網站模組必須瞭解如何與 Portal Framework 互動，才能參與所提供的轉換、編輯、快取與角色架構安全性服務。

入口網站模組的執行方式如同標準的 ASP.NET 使用者控制項。讓使用者控制項在 Portal Framework 內產生效用的必要邏輯，是在特殊的基底類別中進行封裝：PortalModuleControl。只要製作繼承自 PortalModuleControl 類別的使用者控制項，即可建立自訂入口網站模組。

Hello World

範例：在此舉出一個簡單的 Hello World 入口網站模組 (HelloWorld.ascx)。
  
    <%@ Control Inherits="PortalModuleControl" %>

    Hello World!
  在入口網站模組中使用 CSS 樣式

Portal Framework 中包含 Portal.css 的樣式表，可以套用至入口網站中的所有索引標籤。您可以使用 Portal.css 中定義的樣式，使自訂模組與內建模組的外觀一致。最常使用的 Normal 樣式，可以套用到由模組轉換的多數文字上。在此我們已經將 HelloWorld.ascx 更新為使用樣式。
  
    <%@ Control Inherits="PortalModuleControl" %>

    <span class="Normal">Hello World!</span>
  將標題加入模組

在入口網站系統管理員將模組的執行個體加入至入口網站時，可以替此個體下一個描述性的標題。如果要讓模組像標準入口網站模組一樣顯示標題，請將 DesktopModuleTitle 使用者控制項加入模組。DesktopModuleTitle 使用者控制項知道如何從 Portal Framework 取得模組的標題，並轉換成與入口網站其他部分一致的形式。

  
    <%@ Control Inherits="PortalModuleControl" %>
    <%@ Register TagPrefix="Portal" TagName="Title" Src="~/DesktopModuleTitle.ascx" %>

    <portal:title runat="server" />
    <span class="Normal">Hello World!</span>
  為編輯頁加入支援

如果模組有編輯頁，您也可以使用 DesktopModuleTitle 轉換模組標題旁的編輯項目連結。EditText 便是在連結中顯示的文字，而 EditUrl 則是編輯頁相對於入口網站根目錄的路徑。
  
    <%@ Control Inherits="PortalModuleControl" %>
    <%@ Register TagPrefix="Portal" TagName="Title" Src="~/DesktopModuleTitle.ascx" %>

    <portal:title EditText="Edit" EditUrl="PortalModules/HelloWorld/EditHello.aspx" runat="server" />
    <span class="Normal">Hello World!</span>
  建立使用資料的模組

許多入口網站模組將資料庫或 XML 檔案中的資料顯示為其轉換輸出的部分。若要顯示這些資料，您必須像在使用一般 ASP.NET 使用者控制項一樣將其載入並繫結。以下列出使用 Northwind SQL 資料庫的典型範例。
  
    <%@ Control Inherits="PortalModuleControl" %>
    <%@ Register TagPrefix="Portal" TagName="Title" Src="~/DesktopModuleTitle.ascx" %>
    <%@ Import Namespace="System.Data" %>
    <%@ Import Namespace="System.Data.SqlClient" %>

    <script language="VB" runat="server">

        Sub Page_Load(sender As Object, e As EventArgs)

            SqlConnection connection = new SqlConnection("server=localhost;uid=sa;pwd=;database=northwind")
            SqlCommand command = new SqlCommand("SELECT TOP 10 ProductName, UnitPrice FROM Products ORDER BY UnitPrice DESC", connection)

            connection.Open()

            grid1.DataSource = command.ExecuteReader(CommandBehavior.CloseConnection)
            grid1.DataBind()

        End Sub

    </script>

    <portal:title runat="server" />

    <asp:DataGrid id=grid1 Cssclass="Normal" ShowHeader="false" BorderWidth="0" runat="server"/>
 

將自訂模組加入系統管理工具
若要將模組加入入口網站，請直接存取「入口網站系統管理工具」，這項工具內含將新模組動態載入入口網站系統的圖形使用者介面 (GUI) 支援。
