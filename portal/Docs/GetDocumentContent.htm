GetDocumentContent 預存程序

說明：

此預存程序傳回存放在資料庫中文件的 Content、ContentType 及ContentSize。輸入參數為文件的 ItemID。

定義： 
        
   CREATE PROCEDURE Portal_GetDocumentContent
   (
       @ItemID int
   )
   AS
   
   SELECT
       Content,
       ContentType,
       ContentSize,
       FileFriendlyName
   
   FROM Portal_Documents
   
   WHERE
       ItemID = @ItemID
使用的資料庫資料表：

Documents：如 Documents Portal 模組所示，Documents 資料表中的每筆記錄都是單一項目。由於所有的 Documents 模組都將其記錄存放在此資料表中，因此每個項目都包含 ModuleID 以容許在單一查詢中擷取相關項目。

此資料表的主索引鍵為 ItemID 識別欄位。
