AddRole 預存程序

說明：

此預存程序針對特定入口，將新的角色加入資料庫。輸入參數包括 PortalID 與 RoleName。輸出參數則是新資料庫記錄的 RoleID。

定義： 
    
   CREATE PROCEDURE Portal_AddRole
   (
       @PortalID    int,
       @RoleName    nvarchar(50),
       @RoleID      int OUTPUT
   )
   AS
   
   INSERT INTO Portal_Roles
   (
       PortalID,
       RoleName
   )
   
   VALUES
   (
       @PortalID,
       @RoleName
   )
   
   SELECT
       @RoleID = @@Identity
使用的資料庫資料表：

Roles： Roles 資料表中的每項記錄在選擇的入口中會定義一項唯一角色，並透過 UserRoles 查詢值資料表，將使用者加入角色。

此資料表的主索引鍵為 RoleID 識別欄位。
