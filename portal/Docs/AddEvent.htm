AddEvent 預存程序

說明：

此預存程序針對入口中的特定模組，將新事件加入資料庫中。輸入參數包括標題、位置、到期日與事件的說明。輸出參數則是新資料庫記錄的 ItemId。

定義： 
    
   CREATE PROCEDURE Portal_AddEvent
   (
       @ModuleID    int,
       @UserName    nvarchar(100),
       @Title       nvarchar(100),
       @ExpireDate  DateTime,
       @Description nvarchar(2000),
       @WhereWhen   nvarchar(100),
       @ItemID      int OUTPUT
   )
   AS
   
   INSERT INTO Portal_Events
   (
       ModuleID,
       CreatedByUser,
       Title,
       CreatedDate,
       ExpireDate,
       Description,
       WhereWhen
   )
   
   VALUES
   (
       @ModuleID,
       @UserName,
       @Title,
       GetDate(),
       @ExpireDate,
       @Description,
       @WhereWhen
   )
   
   SELECT
       @ItemID = @@Identity
使用的資料庫資料表：

Events： 如 Events Portal 模組所示，Events 資料表中的每項記錄都是單一項目。由於所有的 Events 模組都將其記錄存放在此資料表中，因此每個項目都包含 ModuleID 以容許在單一查詢中擷取相關的項目。

此資料表的主索引鍵為 ItemID 識別欄位。請注意事件說明不得超過 2000 個字元。
