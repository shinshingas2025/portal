DeleteUserRole 預存程序

說明：

此預存程序從指定的安全性角色刪除使用者。Roles 資料表中以個別的入口網站為基準來定義角色。

定義： 
    
   CREATE PROCEDURE Portal_DeleteUserRole
   (
       @UserID int,
       @RoleID int
   )
   AS
   
   DELETE FROM Portal_UserRoles
   
   WHERE
       UserID=@UserID
       AND
       RoleID=@RoleID
使用的資料庫資料表：

UserRoles： UserRoles 資料表提供入口網站安全性角色 (定義於 Roles 資料表) 與使用者 (定義於 Users 資料表) 之間的多對多連接。 在使用 UserRoles 資料表時，每名使用者都能屬於多種角色，而且每項角色的成員也可以為多名使用者。

UserRoles 資料表沒有主索引鍵。
