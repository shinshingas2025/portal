AddLink 預存程序

說明：

此預存程序針對入口中的特定 Links 模組，將新連結加入討論。輸入參數包括標題、URL、檢視順序及連結的說明。輸出參數則是新資料庫記錄的 ItemId。

定義： 
    
   CREATE PROCEDURE Portal_AddLink
   (
       @ModuleID    int,
       @UserName    nvarchar(100),
       @Title       nvarchar(100),
       @Url         nvarchar(250),
       @MobileUrl   nvarchar(250),
       @ViewOrder   int,
       @Description nvarchar(2000),
       @ItemID      int OUTPUT
   )
   AS
   
   INSERT INTO Portal_Links
   (
       ModuleID,
       CreatedByUser,
       CreatedDate,
       Title,
       Url,
       MobileUrl,
       ViewOrder,
       Description
   )
   VALUES
   (
       @ModuleID,
       @UserName,
       GetDate(),
       @Title,
       @Url,
       @MobileUrl,
       @ViewOrder,
       @Description
   )
   
   SELECT
       @ItemID = @@Identity
使用的資料庫資料表：

Links： 如 Links Portal 模組所示，每個 Links 資料表中的記錄都是超連結。由於所有的 Links 模組都將其記錄存放在此資料表中，因此每個項目都包含 ModuleID 以容許在單一查詢中擷取相關項目。

此資料表的主索引鍵為 ItemID 識別欄位。請注意連結說明不得超過 4000 個字元。
