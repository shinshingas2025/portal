GetHtmlText 預存程序

說明：

此預存程序傳回指定 HTML 模組的桌面 HTML 與 Mobile 文字。輸入參數為 ModuleID。

定義： 

   CREATE PROCEDURE Portal_GetHtmlText
   (
       @ModuleID int
   )
   AS
   
   SELECT
       *
   
   FROM Portal_HtmlText
   
   WHERE
       ModuleID = @ModuleID
使用的資料庫資料表：

HtmlText： HtmlText 中的每項記錄都含有與特定 HTML 模組關聯的 HTML 與文字。各項文字欄位如 DesktopHtml、 MobileSummary 與 MobileDetails，都存放在 SQL Unicode 文字欄位中，並能容納多達 1,073,741,823 個字元。

此資料表的主索引鍵為 ModuleID 欄位。
