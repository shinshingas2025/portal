AddUser 預存程序

說明：

此預存程序將新使用者加入資料庫。輸出參數包括「名稱」、「電子郵件地址」及「密碼」。

定義： 
    
   CREATE PROCEDURE Portal_AddUser
   (
       @Name     nvarchar(50),
       @Email    nvarchar(100),
       @Password nvarchar(50),
       @UserID   int OUTPUT
   )
   AS
   
   INSERT INTO Portal_Users
   (
       Name,
       Email,
       Password
   )
   
   VALUES
   (
       @Name,
       @Email,
       @Password
   )
   
   SELECT
       @UserID = @@Identity
使用的資料庫資料表：

Users： Users 資料表中的每項記錄都是唯一使用者識別項。此資料表的主索引鍵為 UserID 識別欄位。
